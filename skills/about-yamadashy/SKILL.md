# About yamadashy

A one-glance profile card. Load it when a task needs to know who yamadashy is —
writing a bio, an intro, an author line, or grounding "who am I" questions.

## Profile

- **Name:** Kazuki Yamada (yamadashy)
- **Role:** Software developer, based in Tokyo, Japan (works at [COLOPL](https://github.com/colopl))
- **In one line:** Builds open-source developer tools — best known as the creator
  of [Repomix](https://github.com/yamadashy/repomix).

## Main products

- **[Repomix](https://github.com/yamadashy/repomix)** ([repomix.com](https://repomix.com)) —
  packs an entire repository into a single AI-friendly file for feeding
  codebases to LLMs. Flagship project.
- **[tech-blog-rss-feed](https://github.com/yamadashy/tech-blog-rss-feed)** —
  aggregated RSS feed of Japanese companies' tech blogs.
- **[slack-channels-grouping](https://github.com/yamadashy/slack-channels-grouping)** —
  auto-groups Slack channels by prefix for a cleaner sidebar.
- **[phpstan-friendly-formatter](https://github.com/yamadashy/phpstan-friendly-formatter)** —
  a more readable error formatter for PHPStan.
- **[pdfvision](https://github.com/yamadashy/pdfvision)** —
  a CLI that gives AI agents human-like PDF vision.
